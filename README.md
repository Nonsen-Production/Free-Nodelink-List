# Free NodeLink

Premium, high-performance public Lavalink nodes list for your audio applications.

## Global Audio Infrastructure

We provide a list of free public Lavalink nodes that are built for speed, stability, and quality.

## Widget

You can embed a live status widget for any node in your markdown or website.

### Usage

Use the following URL format to generate a widget for a specific node:

```
http://free-nodelink.nyxbot.app/api/nodes/widget?host=<NODE_HOST>
```

**Example:**

![Node Widget](http://free-nodelink.nyxbot.app/api/nodes/widget?host=nodelink.lavalink.non-sen.com)

```markdown
![Node Widget](http://free-nodelink.nyxbot.app/api/nodes/widget?host=nodelink.lavalink.non-sen.com)
```

![Widget](http://free-nodelink.nyxbot.app/api/nodes/widget)
To see the full list of nodes, visit [Free NodeLink](http://free-nodelink.nyxbot.app).
